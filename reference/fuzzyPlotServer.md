# Fuzzy enrichment plot server function

Fuzzy enrichment plot server function

## Usage

``` r
fuzzyPlotServer(id, obj, args, config)
```

## Arguments

- id:

  ID string used to match the ID used to call the module UI function

- obj:

  reactive containing fuzzy enrichment object

- args:

  reactive, list with plot arguments, 'numcat' (number of categories to
  plot)

- config:

  reactive list with config settings
