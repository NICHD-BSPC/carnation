# carnation

`carnation` is an interactive Shiny dashboard that makes complex bulk
RNA-Seq data more accessible and intuitive, integrating all facets of
bulk RNA-Seq analysis using three modules - differential expression
analysis, functional enrichment and pattern analysis.

## Details

- Deeply explore analysis results from complex experiments using
  interactive plots.

- Easily keep track of genes of interest using the 'Gene scratchpad'.

- Use fuzzy search to filter and search functional enrichment results.

- Visualize complex patterns using highly customizable gene plot.

- Manage local data in single-user mode or deploy on a server to share
  with collaborators using in-built user management system.

Main function to run the app:
[`run_carnation()`](https://nichd-bspc.github.io/carnation/reference/run_carnation.md)
